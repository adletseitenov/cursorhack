# Поводырь

Веб-приложение-навигатор для незрячих. Телефон висит на груди, камера смотрит вперёд —
VLM анализирует кадры в реальном времени и ведёт человека к цели голосом и вибрацией.

**Live:** https://povodyr.netlify.app

## Как это работает

1. Телефон на груди, задняя камера смотрит вперёд (`getUserMedia`).
2. Сразу после старта камеры ассистент описывает, что впереди (препятствия — в первую
   очередь). Кнопки искать не нужно.
3. Весь экран — кнопка микрофона: коснись и скажи «хочу дойти до бутылки воды» —
   цель извлекается из фразы, начинается ведение. «Стоп» — остановиться,
   «что вокруг» — вернуться к описанию. По прибытии ассистент сам возвращается
   к описанию окружения.
4. Цикл навигации: кадр (768px, JPEG) → VLM через OpenRouter → JSON-команда.
5. Команда озвучивается по-русски (Web Speech API) + вибрация по направлению
   (`hazard` — тревожный паттерн).
6. Строго один запрос в полёте: следующий кадр снимается в момент отправки — латенси
   контура и есть темп ведения.
7. Модель получает последние 3 команды — гасит осцилляцию «лево/право».
8. Judge view: на экране живой лог команд и кадр, который анализирует модель.

## Цели конечного продукта

- [x] **Рабочий голосовой помощник без рук и без зрения**: телефон повесил — и
  разговариваешь. Описание окружения по умолчанию, цель голосом
  («хочу дойти до …»), «стоп», «что вокруг», голосовое подтверждение прибытия.
- [x] **Безопасность прежде всего**: ступени, ямы, препятствия на пути объявляются
  раньше всего; опасность = команда «стоп» + тревожная вибрация; ложный «прибыл»
  отфильтрован (2 тика подряд). Промпты выбраны шутаутом с судейской панелью —
  победил тот, что единственный остановил человека перед лестницей вниз.
- [x] **Темп живого ведения**: латенси контура ~1–1.5 с (тёплая) — команды успевают
  за шагом; модель выбрана латенси-шутаутом через боевой прокси.
- [x] **Живучесть в поле**: потеря камеры/сети не роняет цикл — голосовое
  предупреждение и авто-восстановление; экран не гаснет (wake lock); iOS mute
  switch детектится.
- [x] **Приватность и защита ключа**: ключ OpenRouter только на сервере, прокси
  пропускает только наш формат запросов, кадры нигде не сохраняются.
- [ ] **Полная доступность**: ARIA-разметка для TalkBack/VoiceOver — есть; темп речи
  голосом («говори быстрее/медленнее», запоминается) — есть; постоянный слух без касаний —
  есть (кнопка «Слух», опт-ин: непрерывный микрофон тяжёл для iPhone); резервное
  распознавание голоса через vision-модель (WAV → gemini), если системное сломано — есть;
  впереди: выбор голоса.
- [ ] **Умное зрение**: чтение вывесок, табличек и номеров прямо по курсу — есть;
  впереди: память сцены между кадрами, оценка расстояний по глубине.
- [ ] **Автономность**: офлайн-фолбэк с on-device моделью (WebGPU), мультиязычность,
  GPS-подсказки для улицы.

## Формат ответа модели

```json
{ "instruction": "Два шага вперёд", "direction": "forward", "hazard": false, "arrived": false }
```

## Стек

Фронт: один файл `index.html`, vanilla JS, без сборки.
Бэкенд: одна Netlify Function `/api/vlm` — прокси к OpenRouter (ключ в env, не в браузере).
Камера → canvas → base64 → `/api/vlm` → VLM → Web Speech (голос) + Vibration API +
webkitSpeechRecognition (цель голосом).

## Запуск

Открыть https://povodyr.netlify.app на телефоне (HTTPS обязателен для камеры).
«Старт камеры» — ассистент сразу описывает, что впереди. Коснись экрана в любом
месте и скажи цель: «хочу дойти до двери». Голосовые команды: «стоп», «что вокруг».
Цель можно и вписать в поле текстом.
