# Поводырь

Веб-приложение-навигатор для незрячих. Телефон висит на груди, камера смотрит вперёд —
VLM анализирует кадры в реальном времени и ведёт человека к цели голосом и вибрацией.

**Live:** https://povodyr.netlify.app

## Как это работает

1. Телефон на груди, задняя камера смотрит вперёд (`getUserMedia`).
2. Цикл навигации: кадр (768px, JPEG) → VLM через OpenRouter → JSON-команда.
3. Команда озвучивается по-русски (Web Speech API) + вибрация по направлению
   (`hazard` — тревожный паттерн).
4. Строго один запрос в полёте: следующий кадр снимается в момент отправки — латенси
   контура и есть темп ведения.
5. Модель получает последние 3 команды — гасит осцилляцию «лево/право».
6. Judge view: на экране живой лог команд и кадр, который анализирует модель.

## Формат ответа модели

```json
{ "instruction": "Два шага вперёд", "direction": "forward", "hazard": false, "arrived": false }
```

## Стек

Один файл `index.html`, vanilla JS, без сборки и бэкенда.
Камера → canvas → base64 → OpenRouter (`qwen/qwen3-vl-8b-instruct`) → Web Speech + Vibration API.

## Запуск

Открыть https://povodyr.netlify.app на телефоне (HTTPS обязателен для камеры),
при первом запросе ввести OpenRouter API key (сохраняется в localStorage).
«Старт камеры» → указать цель → «Навигация».
